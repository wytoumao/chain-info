# 🔗 Chain Info Dashboard

实时监控主流交易所的区块链充提状态

## ✨ 功能特性

- 🔄 **实时监控** - 自动每5分钟刷新数据
- 🔍 **智能搜索** - 快速查找目标链
- 🏷️ **分类筛选** - 按UTXO、EVM、EVM L2、Non-EVM分类
- 📱 **响应式设计** - 完美支持桌面端和移动端
- 🎨 **现代UI** - 精美的渐变色彩和动画效果
- 🚀 **高性能** - API缓存和批量请求优化

## 🛠️ 技术栈

- **框架**: Next.js 14
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **部署**: Vercel
- **API**: Gate.io Public API

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 生产运行
npm start
```
